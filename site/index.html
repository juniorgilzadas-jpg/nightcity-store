    🚀 Como Configurar o Bot

    1
    Acesse discord.com/developers/applications → New Application → de um nome ao bot.
    2
    Va em Bot → Add Bot → clique Reset Token e copie o token gerado.
    3
    Em Bot ative: Server Members Intent e Message Content Intent.
    4
    Em OAuth2 → URL Generator: marque bot + permissoes Send Messages e Embed Links. Copie a URL gerada e convide o bot ao seu servidor Discord.
    5
    Online gratis: suba a pasta bot/ no Render.com (plano free) e o site no GitHub Pages. Guia completo no arquivo DEPLOY.md.
    6
    Cole a URL da API (ex: https://seu-bot.onrender.com) no campo abaixo, salve o token no Render e use UptimeRobot para manter o bot acordado 24h (gratis).

    🔑 Token e URL da API

    ⚠ Nunca compartilhe esse token com ninguem!
    Padrao: http://localhost:3001
